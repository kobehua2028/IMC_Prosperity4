# IMC Prosperity 4 Visualizer

[![Build Status](https://github.com/jmerle/imc-prosperity-3-visualizer/workflows/Build/badge.svg)](https://github.com/jmerle/imc-prosperity-3-visualizer/actions/workflows/build.yml)

This repository contains the source code for a visualizer built for the **IMC Prosperity 4** challenge.

It is forked from the original IMC Prosperity 3 visualizer (https://jmerle.github.io/imc-prosperity-3-visualizer/) and has been adapted and extended to support the requirements and data formats of Prosperity 4.

## 🔗 Live Demo

The deployed visualizer is available at:  
https://kevin-fu1.github.io/imc-prosperity-4-visualizer/

## 📌 Background

This project is based on the IMC Prosperity 3 visualizer:  
https://jmerle.github.io/imc-prosperity-3-visualizer/

The original implementation has been modified to:

- Support Prosperity 4 datasets and trading mechanics
- Improve usability and visualization features
- Integrate custom service components for deployment
